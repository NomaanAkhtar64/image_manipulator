## Run Locally

Clone and Install dependencies by:

```bash
  git clone https://github.com/NomaanAkhtar64/image_manipulator
  cd image_manipulator
  npm install
```

Start the server

```bash
  npm run dev
```

Server runs at [localhost](http://localhost:5173)

## Issues
